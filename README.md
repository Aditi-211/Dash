## Installing Dash Car Game

To install **Dash**, follow these steps:

Linux and macOS:

```bash
sudo git clone https://github.com/Aditi-211/Dash.git
```

Windows:

```bash
git clone https://github.com/Aditi-211/Dash.git
```
